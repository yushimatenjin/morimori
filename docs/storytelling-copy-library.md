# Morimorimori 文言指針メモ

このメモは、文言の見た目を揃えるためではなく、
ユーザーが制作を止めないために、文言へ責任を持たせるためのものです。

## 探す章で伝えること

ユーザーは「何を入力すればいいか」「入力後に何が起こるか」を知りたい。

使う文言の方向:
- 生成する地点を選んでください
- 地点を検索して中心を決めます
- この地点を生成対象にします

主 CTA の方向:
- 地点を検索
- 中心地点を確定

## 作る章で伝えること

ユーザーは「今処理中であること」と「待てば進むこと」を知りたい。

使う文言の方向:
- 広域地形を生成しています
- 生成には数十秒から数分かかる場合があります
- 完了後にモデルを保存できます

## 持ち出す章で伝えること

ユーザーは「次に保存すれば完了すること」を知りたい。

使う文言の方向:
- 地形を生成しました。まずモデルを保存してください
- この結果を GLB として出力できます
- 保存後に必要なら配置調整へ進めます

主 CTA の方向:
- 3Dモデルをダウンロード
- GLBを保存

## 仕上げる章で伝えること

ユーザーは「この章は任意であること」を知りたい。

使う文言の方向:
- 必要な場合のみ見え方を調整できます
- 配置と視点は保存後に調整します
- 主導線完了後に仕上げへ進みます

## 例外時に伝えること

エラー文は感情ではなく再開導線を作る。

必須構造:
- 原因: {原因}
- 影響: {影響}
- 次: {次行動}

再試行 CTA の方向:
- 再試行する
- 条件を見直して再実行

## 文言でやってはいけないこと

- 日英を同一画面で混在させる
- 「しばらくお待ちください」だけで終える
- 原因不明のまま再試行だけ促す
- 主 CTA より副操作を強く見せる
